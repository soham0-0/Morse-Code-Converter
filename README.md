## Input Format:
- Use one space to seperate letters and two spaces to seperate words.
- Put the input string inside quotation marks eg. ```"-.- --- -. .. -.-. .... .. .-- .-"```.

## Build:
1. ```git clone https://github.com/soham0-0/Morse-Code-Decoder.git```
2. ```cd Morse-Code-Decoder/```
3. ```make all```
